Ce dépôt contient trois études de cas optimisées en utilisant la programmation linéaire, réalisées dans le cadre du module Recherche Opérationnelle.

1. Situation 6.1 (RIM) : Optimisation du transport de poissons (Nouadhibou - Nouakchott).
2. Situation 6.3 (RIM) :Routage des camions d'eau dans les quartiers périphériques.
